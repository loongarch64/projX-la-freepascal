# projX-la-freepascal

## 项目名称

Freepascal编译器的LoongArch移植

## 支持单位

龙芯中科技术股份有限公司、中国科学院计算技术研究所

## 项目描述

将开源的Free Pascal编译器移植到LoongArch平台。Free Pascal是一个32位和64位专业Pascal编译器，它可以在Intel x86、AMD64/x86-64、PowerPC32/64、SPARC和ARM等多个处理器架构中运行。
